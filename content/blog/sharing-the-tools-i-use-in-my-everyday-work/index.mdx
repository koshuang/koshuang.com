➡️OS：Ubuntu 24.04

我通常都很衝動，基本上 OS 一有更新就會馬上更新，但也因此有時會踩到雷 XD。

手機 OS 基本上也都是一看到通知就立即更新的那種😀

➡️Shell

👉zsh：可互動式的 Shell

👉oh my zsh

👉antigen： zsh plugin manager

👉theme： Powerlevel10k

👉plugins:

- 依據需要自己裝，但我會裝`zsh-autosuggestion`，它會自動提示上次最後一次的指令，方便自動補齊

➡️Terminal: kitty

一個效能很好的 Terminal，是用 Rust 寫的，可以用到 GPU，Render 速度很快。

➡️Tools

👉fzf： 好用的 Command Line 模糊搜尋工具，我很常搭配 Ctrl + R 來模糊搜尋下過的指令紀錄 (History)，快速找到以前執行過的指令。

👉tmux + tmuxinator： 方便管理 Terminal 視窗，可以分成多個 session ，方便快速鍵切換。生產力的好工具！😀

👉vim：預設 git commit message 編輯器，在 rebase 時很方便快速剪下貼上

👉Flameshot：截圖工具

👉DBeaver： Database client

👉tig： 好用的 Git 的 Command Line Tool

➡️Chrome Extensions

👉Fake Filler：方便快速填假資料到表單

👉GoFullPage：方便整頁捲動截圖

👉Tab CtrlP：方便快速搜尋 Tab

➡️ VS Code Plugins

語言相關的就不列了，這邊列一些額外的

👉Dev Containers：方便在 Docker 內開發，尤其開發 Python 非常建議裝

👉PlantUML：方便畫 UML 並自動產生圖

👉Todo Tree

👉markdownlint

👉Gist

👉Code Spell Checker：檢查錯字，加字典

👉change-case：方便轉換各式的格式，像是 snake case, camel case 等
